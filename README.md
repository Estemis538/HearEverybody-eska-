# Переводчик жестов (локальный демо-проект)

Проект запускается локально и работает полностью без БД/хранилища:
камера из браузера -> Python (Flask + OpenCV + MediaPipe) -> распознанное слово -> поле на странице.

## 1) Установка

```bash
cd "/Users/asphen/Desktop/project"
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

## 2) Запуск сервера

```bash
source venv/bin/activate
python app.py
```

Сервер стартует на:
`http://127.0.0.1:5000`

## 3) Проверка в браузере

Открывайте сайт в **обычном системном браузере** (Safari/Chrome/Firefox).

Если у вас появляется ошибка вида `Access to 127.0.0.1 was denied / HTTP 403`, это значит, что вы открываете URL не в системном браузере (например, в изолированном просмотрщике). Решение: откройте ссылку именно в Safari/Chrome/Firefox.

## 4) Использование

На странице включите камеру кнопкой `Включить камеру`, затем покажите одну из 20 базовых поз.

Распознанное слово будет отображаться в текстовом поле.

## Примечание про качество распознавания

Это демо с ограниченным набором 20 жестов; “точность” зависит от освещения, угла руки и того, насколько поза соответствует шаблонам.
